# EMRTS Small Projects

## Projects

### State Capitals
Generates a JSON file of all US state capital addresses, then uses 
geopy to look up and append longitude and latitude for each location.
- [Part 1](state_capitals/state_capitals_part1.py)
- [Part 2](state_capitals/state_capitals_part2.py)

### Water Jug Problem
Solves the classic water jug puzzle using a breadth-first search (BFS) algorithm.
- [Solution](water_jug_problem/water_jug_problem.py)

### FizzBuzz
Classic FizzBuzz implementation in Python. Looping through values and outputting
"Fizz", "Buzz", or "FizzBuzz" depening on set conditions.
- [Solution](fizzbuzz/fizzbuzz.py)

### Scrabble Word Checker
Checks whether a word is valid in Scrabble using the TWL06 word list.
- [Solution](tabind_scrabble/tabind_project.py)

## How to Run
Each project has its own run script. For example:
   cd state_capitals
   ./run_state_capitals.sh

Each project also has a python file by itself if you prefer
a simple python file.
