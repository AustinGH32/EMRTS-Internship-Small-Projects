# FizzBuzz

## Description
A Python program that prints numbers 1-100, replacing multiples of 3 with "Fizz", 
multiples of 5 with "Buzz", and multiples of both with "FizzBuzz".

## Requirements
- Python 3

## How to Run
1. Clone the repo:
   git clone https://github.com/AustinGH32/EMRTS.git
2. Navigate into the folder:
   cd EMRTS/fizzbuzz
3. Run the script:
   ./run_fizzbuzz.sh
