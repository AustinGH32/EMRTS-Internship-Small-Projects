# State Capitals

## Description
Part 1 creates a JSON file of all US state capital addresses.
Part 2 reads that JSON and adds longitude and latitude for each address.

## Requirements
- Python 3
- geopy library:
   pip install geopy

## How to Run
1. Clone the repo:
   git clone https://github.com/AustinGH32/EMRTS.git
2. Navigate into the folder:
   cd EMRTS/state_capitals
3. Run the script:
   ./run_state_capitals.sh
